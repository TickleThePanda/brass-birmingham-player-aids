Brass Birmingham - Player aid

Definitions

Your Network

A Location is in your Network if you have an Industry tile in it or a Link tile next to it.

Connections

A Location is Connected if a route can be traced via Link tiles of any player.

Consuming

Consume Coal

Remove first available from:

  1. Closest connected Coal Mine
  2. Market, if connected to Merchant. Pay cost.

Consume Iron

Remove first available from:

  1. Any Iron Works
  2. Market. Pay cost.

Consume Beer

Remove from choice:

  • Any of your own Brweries
  • Connected opponent's Brewery
  • Connected Merchant beer slot (only using Sell action)

Multiple beer must come from different sources

Actions

Build

  1. Choose and discard appropriate card:
    • Industry cards: must build within Network; build matching Industry (or any if wild)
    • Location cards: doesn't need to be within Network; build in specified location, or any if wild (except farm brewery).
  2. Pick lowest level Industry tile.
  3. Place within Location, using single-icon slots first.
  4. Pay cost and consume required resources.
  5. Produce resources
  6. Check market
    • Sell newly produced Coal, if connected to merchant.
    • Sell newly produced Iron, regardless of connection.

Era restrictions

  • Canal era: only 1 tile allowed per player per Location
  • Rail era: multiple tiles allowed per player per Location

Overbuilding

Over your own tiles:

  • Any kind is allowed.
  • Any cubes are lost.

Over opponents tiles:

  • Only for Coal Mine or Iron Works.
  • Must be cubes of placed Industry type available on the board.

Sell

  1. Discard any card.
  2. Flip 1 tile connected to Merchant with matching icon.
  3. Consume required beer indicated on Industry tile.
  4. If beer is consumed from Merchant, get the Merchant Beer Bonus.
  5. May repeat process from step 2 for other unflipped buildings.

Loan

  2. Take Â£30 from bank
  3. Move Income marker 3 levels down, to highest space within level

Scout

  1. Discard any 3 cards.
  2. Take 1 wild Location card and 1 wild Industry card.

Not allowed if you already have a wild card in your hand.

Develop

  1. Discard any card.
  2. Remove 1 or 2 Player Mat tiles from lowest level of 1 or 2 Industries.
  3. Consume 1 Iron for each tile removed.

Not allowed on Industry tiles with lightbulb symbol

Network

  1. Discard any card.
  2. Place Link tile on empty line next to Location in your Network.
  3. Pay cost and consume required resoruces, depending on current age

If building 2 rail links, the consumed beer must be from brewery, not merchant.
